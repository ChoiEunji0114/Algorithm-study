## 문제
- 백준 11407 : 동전 0
- https://www.acmicpc.net/problem/11047
- 그리디 알고리즘
- 동전 교환

<br/>

## 풀이
- 간단한 그리디 알고리즘 문제.
- 가장 **큰 가치의 동전** 부터 탐색을 시작한다.   
지금 현재 동전합과 탐색 동전합이 작을 경우 현재 값에 탐색 동전합을 뺀 후 ``` ans ``` 에 1을 더한다.   


<br/>

## 코드
```c++
#include <iostream>
#define MAX 100000001

using namespace std;

int main(void){
    
    int n, total, arr[MAX];
    cin >> n >> total;
    
    for(int i=1; i<=n; i++){
        cin >> arr[i];
    }
    
    int ans = 0; // 동전 갯수
    for(int i=n; i>0; i--){
        while(total>=arr[i]){
            total -= arr[i];
            ans++;
        }
    }
    
    cout << ans << endl;
    
    return 0;
}
```
